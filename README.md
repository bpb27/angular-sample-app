# angular-sample-app
Basic Angular App with Tests

## Setup
* `npm install`
* `bower install`
* Replace the Firebase URL in app/services.js and app/controllers.js with your own URL
* Add your own Travis CI badge to the README.md

## Running the app
* `npm start`

## Running the tests
* `npm test`
